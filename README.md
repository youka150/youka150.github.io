# Youka's Blog

A beautiful, minimalist personal blog page designed for GitHub Pages. 

## Features
- **Design Aesthetics:** Modern dark mode UI with soft gradients, glassmorphism (`backdrop-blur`), and tailored typography (Inter font).
- **Magnetic Cursor:** A custom trailing cursor with magnetic interactions on links and cards.
- **Ripple Animations:** Click ripple effects on interactive elements like article cards.
- **GSAP Animations:** Smooth entrance animations and scroll-triggered reveals.
- **Tailwind CSS:** Responsive and clean styling utility framework.

## Setup
Since this is a static site built for GitHub Pages, simply push these files to the `main` branch. GitHub Pages will serve `index.html` directly.

- `index.html` - The main structure
- `styles.css` - Custom CSS animations and cursor styling
- `script.js` - JS logic for GSAP, magnetic cursor, and ripple effects
